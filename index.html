Time Clock Device

Set up a dedicated tablet or shared device where staff can clock in and out. The Time Clock page is separate from the main POS.

  1. Click Generate Code to create a 6-character activation code.
  2. On the device you want to use, open timeclock.html (same URL as this app, different page).
  3. Tap Activate Device on that page and enter the code. The device is now registered.
  4. Staff sign in with their own accounts, not the owner account, to clock in and out.

Activation Code

Generating a new code immediately invalidates the previous one. The device only needs to be activated once.
